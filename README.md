# Agent Fabric

An asynchronous orchestration engine designed to manage multiple AI agents using Ollama.

## Key Features
- **Async Queueing**: Manages tasks concurrently for high efficiency.
- **Robust Error Handling**: Captures and reports agent failures gracefully.
- **Local-First**: Built for privacy and performance using Ollama.

## Setup
1. Clone this repository.
2. Create and activate a virtual environment:
   `python3 -m venv venv && source venv/bin/activate`
3. Install dependencies:
   `pip install -r requirements.txt`
4. Run the application:
   `python3 main.py`
